### ūüĒ≤ IoT Blackbox ¬∑ Accident Data Recorder
`Hardware ¬∑ Embedded Systems ¬∑ Forensics`

<br/>

A forensic-grade **two-wheeler black box** that logs collision vectors, detects high-G impact events, and timestamps crash data in real time.

**Stack**
```
MCU      ‚Üí  Arduino / ESP32
Sensor   ‚Üí  MPU6050 (6-axis IMU)
Signal   ‚Üí  Inertial Thresholding Algorithms
Storage  ‚Üí  SD Card + Serial Log
Language ‚Üí  C / Python
```

**Why it matters**
> Road accident investigations in India depend almost entirely on eyewitness accounts. This device gives investigators ground-truth data ‚ÄĒ velocity, impact angle, and timestamp ‚ÄĒ without any human in the loop.

### ūüĒ≤ OpsFlow AI ¬∑ Autonomous Business Ops
`AI ¬∑ Agentic Workflows ¬∑ Full Stack`

<br/>

An **autonomous operations layer** for business teams. OpsFlow replaces manual approvals and reactive decision-making with self-healing AI agents that detect problems and act ‚ÄĒ before humans notice.

**Stack**
```
Frontend  ‚Üí  React + Vite
Backend   ‚Üí  Next.js + Supabase
Workflow  ‚Üí  n8n (Agentic Layer)
Database  ‚Üí  PostgreSQL (Supabase)
Auth      ‚Üí  Row-Level Security
```

**What it does**
> Detects low inventory ‚Üí compares vendors ‚Üí creates purchase orders. Flags overdue payments ‚Üí analyses client history ‚Üí sends reminders. All without a human in the loop unless approval is required.
